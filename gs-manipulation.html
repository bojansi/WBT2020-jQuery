.clone()

jQuery metoda .clone() omogućava kopiranje DOM elemenata.

Koristi se vrlo jednostavno – odaberu se DOM elementi, a onda se nad izabranim skupom elemenata pozove metoda clone().

Metoda za kloniranje je vrlo korisna kada želite da kopirate i premestite ne samo DOM elemente, već i događaje vezane za klonirane DOM elemente.

Sintaksa:

$(selector).clone(true|false)
Kada koristite .clone () metodu, možete da izmenite klonirane elemente ili njihov sadržaj pre (ponovnog) ubacivanja u dokument.

Korišćenje ove metode može izazvati stvaranja elemenata sa duplikatima atributa id, koji bi trebalo da budu jedinstveni. Gde je to moguće, preporučuje se izbegavanje kloniranja elemenata sa ovim atributom ili korišćenje atributa klase kao identifikatora.

.wrap()

Metoda .wrap () se koristi za obavijanje sadržaja, tačnije omogućava da se svaki podudarni element ili elementi u skupu obaviju odabranim HTML elementima.

Sintaksa:

$(selector).wrap(wrappingElement,function(index))
wrappingElement određuje koje HTML element(e) treba obaviti oko svakog izabranog elementa, moguće vrednosti:
  • HTML elementi
  • jQuery objekti
  • DOM elementi

.wrapAll()

Metoda .wrapAll() obavija određene HTML elemente oko svih izabranih elemenata.

Sintaksa:

$(selector).wrapAll(wrappingElement,function(index))
wrappingElement određuje koje HTML element(e) treba obaviti oko izabranih elemenata, moguće vrednosti:
  • HTML elementi
  • jQuery objekti
  • DOM elementi

.wrapInner()

Metoda .wrapInner() pravi novi element i smešta ga kao podređeni element selektovanom elementu, koji okružuje sve elemente unutar njega.

Sintaksa:

$(selector).wrapInner(wrappingElement,function(index))
wrappingElement određuje koje HTML elemente treba obaviti oko sadržaja svakog izabranog elementa, moguće vrednosti:
  • HTML elementi
  • jQuery objekti
  • DOM elementi

.append()

Metoda .append () pravi novi objekat prema zaditim parametrima i smešta ga na kraj seta elemenata unutar selektovanog elementa.

Sintaksa:

$(selector).append(content,function(index,html))
content sadržaj za umetanje, moguće vrednosti:
  • HTML elementi
  • jQuery objekti
  • DOM elementi
function(index,html) određuje funkciju koja vraća sadržaj za umetanje
index-vraća položaj elementa u skupu

Metoda .append() ubacuje navedeni sadržaj kao poslednji podređeni element (last child) svakog elementa u kolekciji jQuery, da biste ga umetnuli kao prvi podređeni element(first child), koristite .prepend ().

Metode .append() i .appendTo() izvršavaju isti zadatak. Glavna razlika je u sintaksi, metoda .appendTo() selektovani element dodaje na kraj seta elemenata unutar prosleđenog elementa.

.remove()

Briše selektovani element i sve njegove podređene elemente(child) iz DOM-a. Moguće je proslediti parametar za odabir elemenata za brisanje.

Sintaksa:

$(selector).remove()

Koristite .remove () kada želite da uklonite sam element, kao i sve u njemu. Pored samih elemenata, uklanjaju se svi povezani događaji i podaci. Da biste uklonili elemente bez uklanjanja podataka i događaja, koristite .detach ().

.empty()

Slično .remove() metodi, .empty() metoda briše sve child elemente u selektovanom elementu iz DOM-a

Sintaksa:

$(selector).empty()
Ovom metodom uklanjaju se ne samo podređeni elementi (i ostali potomci), već i sav tekst unutar skupa podudarnih elemenata. To je zato što se, prema DOM specifikaciji, bilo koji niz teksta u elementu smatra podređenim čvorom tog elementa.

.replaceAll()

Metod .replaceAll() zamenjuje selektovane elemente novim HTML elementima.

Sintaksa:

$(content).replaceAll(selector)
content određuje sadržaj za umetanje
selector određuje koje elemente treba zameniti

.replaceWith()

Metoda replaceWith() zamenjuje selektovane elemente novim sadržajem.

Sintaksa:

$(selector).replaceWith(content,function(index))
content određuje sadržaj za umetanje, moguće vrednosti:
  • HTML elementi
  • jQuery objekti
  • DOM elementi
