Preuzimanje jQuery-a

Za preuzimanje jQuery-ja dostupne su kompresovane i nekompresovane kopije jQuery datoteka. Nekompresovane datoteke se najpre koriste tokom razvoja ili otklanjanja grešaka. Kompresovana datoteka je manja od nekompresovane. Ona štedi propusni opseg i poboljšava performanse u proizvodnji zbog čega se ovaj vid datoteke češće postavlja na sajt.

Da biste lokalno preuzeli ove datoteke, pritisnite desnim tasterom miša na vezu i u meniju izaberite "Sačuvaj kao.."

Menadžeri paketa

jQuery je registrovan na menadžeru paketa za JavaScript programski jezik. jQuery se može instalirati pomoću sledećih menadžera paketa:

Preuzimanje jQuery-a korišćenjem npm

# npm install jquery

Preuzimanje jQuery-a korišćenjem Yarn-a

# yarn add jquery

Pomoću ovih komandi instaliraće se jQuery u node_modules direktorijumu.

Preuzimanje jQuery-a korišćenjem Bower-a

# bower install jquery

Pomoću ove komande ćemo instalirati jQuery u bower_components direktorijumu.

Korišćenjem Bower-a postoji mogućnost instalacije jQuery-a sa jednog od gore navedenih URL-ova umesto iz registrovanog paketa. Na primer ako želite instalirati samo nekompresovanu datoteku jQuery-a možete instalirati samo tu datoteku sledećom naredbom:

# bower install https://code.jquery.com/jquery-3.5.1.js

Korišćenje jQuery-a pomoću CDN-a

CDN nudi poboljšanje performansi hostovanjem jQuery-a na serverima širmo sveta. Ovo takođe predstavlja prednost ako je posetilac web strane već preuzeo kopiju jQuery-a sa istog CDN-a jer neće morati ponovo da ga preuzima. Da bi koristili jQuery CDN potrebno je referencirati datoteku u script tagu direktno sa jQuery CDN domena.
