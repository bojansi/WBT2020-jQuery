Šta je to jQuery?

jQuery je mala, brza i moćna JavaScript biblioteka.
Pomoću njega možete da pišete moćne aplikacije u JavaScriptu i kreirate animirane efekte.
Između ostalog, jQuery je dobar i za kreiranje XML(Ajax) upita, manipulaciju DOM-om, pravljenje menija sa podmenijem, kao i kreiranje drag & drop interfejsa.

jQuery je promenio način na koji milioni ljudi pišu JavaScript

 

Zašto jQuery?

Otvorenog je koda, i reč je o projektu sa licencama MIT i GNU General Public License (GPL).
Neverovatno je popularna, što znači da je korisnička zajednica ogromna i da ima mnogo onih koji pružaju pomoć kao programeri i/ili instruktori.
Namenski je napravljena tako da se izbegnu konflikti s drugim JavaScript bibliotekama.
Razvija se otvoreno, što znači da svako može doprineti ispravljanju grešaka, unapređenju i razvoju.

Filozofija biblioteke jQuery

Filozofija ove biblioteke je „Piši manje, uradi više“ (engl. write less, do more), koja se može dalje razložiti u tri koncepta:

  • Nalaženje određenih elemenata (pomoću CSS selektora) i njihova upotreba na neki način (pomoću jQuery metoda)
  • Ulančavanje više jQuery metoda radi primene nad skupom elemenata
  • Primena jQuery omotača (engl. wrapper) i posredne (implicitne) iteracije

Da li će jQuery raditi na svim veb čitačima?

Tim koji radi na njegovom razvoju poznaje sve probleme koji mogu nastati na različitim čitačima, stoga su ugradili rešenja za svaki od njih unutar jQuery biblioteke.
jQuery ima podršku za sve glavne čitače uključujući Internet Explorer 6, a testira se i optimizuje za razvoj na modernim čitačima (Chrome 1, Chrome Nightly, IE 6, IE 7, IE 8, Opera 9.6, Safari 3.2, WebKit Nightly, Firefox 2, Firefox 3, Firefox Nightly).

Povezani projekti

jQuery Mobile postavlja slogan „piši manje, uradi više“ na viši nivo: Omogućava dizajniranje odgovarajuće veb lokacije ili aplikacije koja će raditi na svim popularnim platformama za pametne telefone, tablete i računare.

jQuery Mobile

jQuery UI je kolekcija GUI dodataka, animiranih vizuelnih efekata i tema implementiranih sa jQuery bibliotekom i HTML-om. Pri formiranju veb aplikacije, jQuery UI je savršen izbor.

QUnit je okruženje koje služi za testiranje JavaScript kôdova. Koristio se u jQuery projektu za testiranje jQuery, jQuery UI i jQuery Mobile tako da je to okruženje generisano da testira bilo koji JavaScript kôd.

QUnit
