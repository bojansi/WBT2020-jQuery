.show()

Služi za prikazivanje sakrivenih ili tek kreiranih elemenata na stranici.

Razliciti nacini da se pozove funkcija show()

- Bez ikakvih argumenata, odmah prikazuje element(bez ikakve animacije)

.show()

- Odredjenom duzinom trajanja

.show(trajanje animacije)

- Odredjenom duzinom trajanja i odredjenom tranzicijom

.show(trajanje animacije, tip tranzicije)

- Dodavanje klase ili vise klasa pomocu funkcije

.show({...opcije...})

U opcije pored gore pomenutih svojstava duration(trajanje animacije) i easing(tip tranzicije) moguce je odrediti jos neke dodatne parametre kao sto su:

  • start - funkcija koja se poziva kada animacija pocne
  • done - funkcija koja se poziva kada se animacija zavrsi
  • fail - funkcija koja se poziva kada se animacija ne izvrsi uspesno

.animate()

Metoda animate() omogućava nam da kreiramo efekte animacije na bilo kojem numeričkom svojstvu CSS-a. Jedini potreban parametar je običan objekat CSS svojstava. Ovaj objekat je sličan onom koji se može poslati .css() metodi.

Razliciti nacini da se pozove funkcija animate()

- Sa objektom CSS svojstava(pored toga moze se definisati i duzina trajanja animacije i tip tranzicije)

.animate(CSSsvojstva, [trajanje], [tranzicija])

- Sa objektom CSS svojstava i dodatnim opcijama

.animate(CSSsvojstva, {...opcije...})

U opcije pored gore pomenutih svojstava duration(trajanje)b> i easing(tip tranzicije) moguce je odrediti jos neke dodatne parametre kao sto su:

  • start - funkcija koja se poziva kada animacija pocne
  • done - funkcija koja se poziva kada se animacija zavrsi
  • fail - funkcija koja se poziva kada se animacija ne izvrsi uspesno
